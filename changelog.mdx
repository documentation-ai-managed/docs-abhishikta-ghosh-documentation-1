## How to Contribute

Follow these steps to suggest changes or report issues.

<Steps>
  <Step title="Check Existing Issues" icon="search">
    Review the [GitHub issues](https://github.com/user/abhishikta-ghosh-docs/issues) for duplicates.
  </Step>
  <Step title="Submit a Pull Request" icon="git-pull-request">
    Fork the repo, make changes, and open a PR with clear description.
  </Step>
  <Step title="Test Locally" icon="test-tube">
    <CodeGroup tabs="npm,yarn">
```bash
npm install
npm run dev
```
```bash
yarn install
yarn dev
```
    </CodeGroup>
  </Step>
</Steps>

<Tabs>
  <Tab title="Features" icon="star">
    Propose new components like Video embeds.
  </Tab>
  <Tab title="Bug Fixes" icon="bug">
    Report parsing errors with exact reproduction steps.
  </Tab>
</Tabs>
